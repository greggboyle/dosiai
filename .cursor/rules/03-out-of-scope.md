# Out of scope for v1 (do not build, do not "helpfully" implement)

These are explicitly deferred. Cursor will frequently try to "helpfully" implement these; refuse.

- CRM integration (Salesforce, HubSpot, etc.) — manual win/loss only in v1
- Multi-language support — English only
- Bring-your-own-key (BYOK) for AI vendors — platform-managed keys only
- Native mobile apps — web responsive only
- Sentiment analysis on social media at scale beyond what AI vendors return
- Custom scraper infrastructure for non-AI sources
- Predictive forecasting of competitor moves
- Low-confidence review queue (low-relevance review queue is in scope; low-confidence is deferred)
- Enterprise tier features (SSO/SCIM, BYOK, custom data residency, sub-hourly sweeps) — placeholder only
